# capstone-project-iron-triangle

## Installation
### Backend
cd backend/app  
pip install -r requirements.txt

### Frontend
cd frontend/material-kit-react-master  
npm install

## Running
### Backend
cd backend/app/demo  
python \__init__.py

### Frontend
cd frontend/material-kit-react-master  
npm start  
  
Browse http://127.0.0.1:5000/static/swagger-ui/index.html
